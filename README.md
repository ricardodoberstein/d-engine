# D-Engine

This project is made for learning purpuses, please make sure you know what you are doing if you plan to fork or use this code base.
## Requirements
* Ubuntu OS
* CMake
* [Allegro](https://liballeg.org/)

## Compilation

```bash
./build.sh
```

## Running tests

```bash
./test.sh
```
